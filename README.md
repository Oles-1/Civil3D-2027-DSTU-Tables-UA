# Civil 3D ДСТУ Таблиці (Відомість кутів повороту)

Плагін для Autodesk Civil 3D (протестовано на версії 2027), який автоматизує створення "Відомості кутів повороту, прямих і кривих" для проектування лінійних споруд. Та надає швидкий доступ до інструментів розрахунку довжин і площ, швидкого і зручного копіювання значень, обереня по слоях виділеного ітд.

Оформлення таблиць суворо відповідає вимогам **ДСТУ Б А.2.4-29:2008** (Автомобільні дороги. Земляне полотно і дорожній одяг. Робочі креслення).

## 🚀 Можливості плагіна

- Автоматичний збір даних з вибраної траси (Alignment).
- Генерація двох видів таблиць прямо в робочому просторі Excel:
  - **Базова відомість** (18 стовпців)
  - **Повна відомість** (24 стовпці, з урахуванням перехідних кривих)
- Калькулятор сум:
  -Інструмент для швидкого підрахунку довжин та площ безпосередньо в кресленні.
  -Швидке і зручне копіювання значень.
  -Обирання сум в обрагних слоях.
- Власний графічний інтерфейс (вкладка "+ІНСТРУМЕНТИ" на стрічці Ribbon) зі зручними кнопками.

## 📥 Встановлення (без NETLOAD)

Встановлення відбувається через механізм Autoloader, що не вимагає постійного ручного підвантаження.

1. Перейдіть на сторінку Releases цього репозиторію.
2. Завантажте останній випущений файл (архів .zip), натиснувши на посилання Source code (zip) або на ваш підготовлений архів у розділі "Assets".
3. Розпакуйте архів та скопіюйте папку `Oles_Civil_up.bundle` за одним із шляхів:
   - Тільки для вашого користувача Windows: `%appdata%\Autodesk\ApplicationPlugins`
   - Для всіх користувачів комп'ютера: `C:\ProgramData\Autodesk\ApplicationPlugins`
4. Перезапустіть Civil 3D.
5. На стрічці меню автоматично з'явиться нова вкладка з інструментами.

## 🛠 Використання

1. Відкрийте креслення Civil 3D із побудованою трасою.
2. Перейдіть на вкладку "+ІНСТРУМЕНТИ".
3. Натисніть кнопку потрібної відомості (на 18 або 24 стовпці).
4. Виберіть трасу на кресленні.
5. Зачекайте кілька секунд — плагін самостійно відкриє Excel, заповнить усі дані, вирівняє текст та накреслить рамки.
6. Для калькулятора: Натисніть кнопку "Калькулятор сум" для швидких розрахунків.
7. Зявится вікно. з значенями і кнопками.
